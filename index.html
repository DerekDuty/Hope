Welcome to my practice web site. It is written in HTML.

It is important that all of my paragraphs have paragraph tags around them, so I can keep my content organized and have control over the elements that make up my pages. The paragraph tags have predetermined properties defined by the browser such as a 5 px space above and below them. These properties can be manipulated with CSS but currently are not.
